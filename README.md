### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/romirm/pennapps.git
   cd pennapps
   ```

2. **Install dependencies**
   ```bash
   # For Node.js projects
   npm install
   # or
   yarn install
   
   # For Python projects
   pip install -r requirements.txt
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start the development server**
   ```bash
   # Frontend
   npm run dev
   # or
   yarn dev
   
   # Backend (if separate)
   npm run server
   # or
   python app.py
   ```

5. **Open your browser**
   Navigate to `http://localhost:3000` (or your configured port)
